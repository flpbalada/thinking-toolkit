# Thinking Toolkit

A curated marketplace of Claude Code plugins for product managers and critical
thinkers. Structured methodologies to analyze problems, challenge assumptions,
and make better decisions.

## What's Inside

| Plugin                       | Description                                                           |
| ---------------------------- | --------------------------------------------------------------------- |
| **Five-Whys**                | Root cause analysis using the Five Whys technique                     |
| **Business-Model-Canvas**    | Strategic business model design using the Business Model Canvas       |
| **Cognitive-Biases**         | Understanding and applying cognitive biases in product design         |
| **Cognitive-Fluency**        | Apply cognitive fluency principles to improve clarity and trust       |
| **Cognitive-Load**           | Design experiences that optimize mental resources                     |
| **Curiosity-Gap**            | Create engagement through strategic information gaps                  |
| **Fogg-Behavior-Model**      | Design behavior change using B=MAP (Motivation, Ability, Prompt)      |
| **Game-Theory-Tit-for-Tat**  | Apply Tit for Tat strategy for negotiations and repeated interactions |
| **Graph-Thinking**           | Non-linear problem solving using graph-based mental models            |
| **Halo-Effect-Psychology**   | Apply the halo effect in product design, UX, and first impressions    |
| **Hicks-Law**                | Optimize decision-making speed by managing choice quantity            |
| **Hooked-Model**             | Design habit-forming products using the Hook cycle                    |
| **Hypothesis-Tree**          | Structure complex questions into testable hypotheses using MECE       |
| **Jobs-to-be-Done**          | Understand customer motivations through job theory                    |
| **Kanban**                   | Visualize and optimize workflow with Kanban boards and WIP limits     |
| **Loss-Aversion-Psychology** | Leverage loss aversion in product design, messaging, and retention    |
| **Making-Product-Decisions** | Framework for structured product decision-making                      |
| **Progressive-Disclosure**   | Reduce complexity by revealing information progressively              |
| **Self-Initiated-Triggers**  | Design internal triggers for sustained user engagement                |
| **Skill-Creator**            | Guide for creating Claude Code custom skills                          |

## Why Use It

- **Structured thinking** — proven frameworks instead of ad-hoc analysis
- **Ready to use** — no setup, just install and go
- **PM-focused** — built for product managers and decision makers

## Quick Start

**1. Add the marketplace**

```bash
/plugin marketplace add flpbalada/thinking-toolkit
```

**2. Install plugins**

```bash
# Install all plugins
/plugin install thinking-toolkit

# Or install specific plugin
/plugin install thinking-toolkit:five-whys
```

## Requirements

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) CLI installed
